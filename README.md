# Python Adventure Game (Using NumPy)

A text-based Python console adventure game where the player explores paths, makes choices, and faces challenges.  
This project demonstrates beginner-friendly Python programming while using *NumPy arrays* to manage game options and paths efficiently.

## How to Play

1. Save game-using-numpy.py on your computer.  
2. Open Command Prompt (Windows) or Terminal (Mac/Linux) and navigate to the folder containing the file.  
3. Run the program:

```bash
python game-using-numpy.py
